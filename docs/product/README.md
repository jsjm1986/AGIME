## 快速开始

AGIME 提供了开箱即用的安装包，无需复杂配置。

[下载最新版本](https://github.com/jsjm1986/AGIME/releases)

- **Windows**: `AGIME-Setup-x.x.x.exe`
- **macOS**: `AGIME-x.x.x.dmg`
- **Linux**: `AGIME-x.x.x.AppImage` / `.deb`
