### 数据安全流转

1. **发送请求**: 员工通过加密通道发送任务。
2. **权限校验**: Server 校验 RBAC 权限。
3. **隐私脱敏**: 自动识别并替换 PII (个人敏感信息)。
4. **本地推理**: 任务在内网服务器/本地电脑完成。
5. **审计归档**:所有操作日志留痕不可篡改。

### 访问控制

- **RBAC**: 基于角色的权限访问控制 (Owner / Admin / Member)。
- **LDAP/SSO**: 支持企业统一身份认证 (定制版)。
- **资源隔离**: 部门间知识库和技能完全隔离。

## 部署方案

### 1. LAN 模式 (轻量级)
- **适用**: 10-50 人中小团队
- **架构**: 无中心服务器，P2P 直连
- **成本**: **0** (使用现有 PC)

### 2. Cloud 模式 (标准级)
- **适用**: 50-500 人中型企业/跨地域团队
- **架构**: 需一台 Linux 服务器 (4核/16G起)
- **部署**: Docker / Systemd 

### 3. 定制部署 (企业级)
- **适用**: 500+ 人大型集团
- **支持**: 私有云/混合云，高可用集群
- **服务**: 定制开发，SLA 保障
