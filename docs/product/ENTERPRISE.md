## 📦 部署方案

### 方案一：团队版（LAN 模式）

```
适用：10-50 人团队
部署：局域网内，无需云服务器
成本：免费
```

**快速启动**：
1. 团队负责人安装 AGIME
2. 启用 LAN 模式
3. 生成邀请链接分享给团队
4. 成员加入后即可协作

### 方案二：企业版（Cloud 模式）

```
适用：50+ 人团队 / 多办公地点
部署：内网服务器或私有云
成本：服务器成本
```

**部署步骤**：
```bash
# 1. 下载 Linux 服务端
wget https://github.com/jsjm1986/AGIME/releases/latest/download/AGIME-linux-x64.tar.gz

# 2. 解压并启动
tar -xzf AGIME-linux-x64.tar.gz
cd AGIME-linux-x64/resources/bin
./agimed agent

# 3. 配置为系统服务（可选）
# 4. 客户端连接服务器地址
```

### 方案三：定制开发

```
适用：特殊业务需求
内容：定制功能、系统集成、专属培训
咨询：微信 agimeme
```
