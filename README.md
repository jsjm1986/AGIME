## ✨ 核心功能

### 🤖 智能对话
用自然语言交流，像和助理说话一样。支持上下文记忆，理解你的意图。

### 🔄 食谱系统（工作流复用）
把一次成功的对话保存为"食谱"，下次一键重复执行。

```
场景：每月需要生成相同格式的报告
→ 第一次让 AGIME 帮你做，保存为食谱
→ 以后每月点击执行，自动完成
```

### ⏱️ 定时调度
设置任务自动执行，每小时、每天、每周、每月，灵活配置。

### 🧩 扩展系统（MCP 插件）
通过插件无限扩展能力：

| 插件 | 功能 |
|------|------|
| **Developer** | 代码分析、文件编辑、命令执行 |
| **ComputerController** | 自动化操作电脑 |
| **Memory** | 记住你的偏好和习惯 |
| **Playwright** | 自动浏览网页、填表、截图 |

### 🧠 智能记忆
AGIME 会记住你的偏好，越用越懂你。

### ⚡ 四种工作模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 🟢 **自主模式** | AI 自由操作 | 信任的重复任务 |
| 🟡 **智能模式** | 高风险操作需确认 | 日常使用推荐 |
| 🔴 **手动模式** | 每步都需确认 | 敏感操作 |
| ⚪ **聊天模式** | 仅对话，不执行 | 咨询问答 |

## 👥 团队协作系统 (v2.5.0-team 新增)

AGIME 现在支持**团队协作**功能，让你可以与团队成员共享 AI 技能、工作流和扩展。

### 🌐 两种协作模式

| 模式 | 说明 | 适用场景 | 特点 |
|------|------|----------|------|
| 🏠 **LAN 模式** | 局域网内直接连接 | 办公室/家庭网络 | 无需服务器，即开即用 |
| ☁️ **Cloud 模式** | 通过云服务器连接 | 远程团队/分布式协作 | 跨地域，需部署服务器 |

### ✨ 资源共享系统

#### 可共享的资源类型

| 资源类型 | 说明 | 示例 |
|----------|------|------|
| 🛠️ **Skills（技能）** | 自定义 AI 能力 | 代码审查技能、文档生成技能 |
| 📋 **Recipes（食谱）** | 工作流模板 | 每日报告生成、数据处理流程 |
| 🔌 **Extensions（扩展）** | MCP 插件配置 | 数据库连接、API 集成 |

#### 🛡️ 资源保护级别

每个共享资源可以设置不同的保护级别：

| 保护级别 | 图标 | 说明 | 权限要求 |
|----------|------|------|----------|
| **Public** | 🌍 | 所有成员可见可用 | 无限制 |
| **Team Only** | 👥 | 仅团队成员可见 | 需要加入团队 |
| **Admin Only** | 🔒 | 仅管理员可用 | 需要 Admin 权限 |
| **Owner Only** | 👑 | 仅创建者可用 | 仅资源拥有者 |

### 👤 成员角色与权限

#### 三级权限体系

| 角色 | 说明 | 权限范围 |
|------|------|----------|
| 👑 **Owner** | 团队创建者/拥有者 | 完全控制权 |
| 🛡️ **Admin** | 管理员 | 管理成员和资源 |
| 👤 **Member** | 普通成员 | 使用和分享资源 |

#### 详细权限矩阵

| 操作 | Owner | Admin | Member |
|------|:-----:|:-----:|:------:|
| **团队管理** | | | |
| 删除团队 | ✅ | ❌ | ❌ |
| 修改团队信息 | ✅ | ✅ | ❌ |
| 查看团队信息 | ✅ | ✅ | ✅ |
| **成员管理** | | | |
| 邀请新成员 | ✅ | ✅ | ❌ |
| 移除成员 | ✅ | ✅* | ❌ |
| 修改成员角色 | ✅ | ❌ | ❌ |
| 离开团队 | ❌ | ✅ | ✅ |
| **资源管理** | | | |
| 分享资源 | ✅ | ✅ | ✅ |
| 删除任何资源 | ✅ | ✅ | ❌ |
| 删除自己的资源 | ✅ | ✅ | ✅ |
| 安装资源到本地 | ✅ | ✅ | ✅ |
| 修改资源保护级别 | ✅ | ✅ | ❌ |

> *Admin 不能移除 Owner 或其他 Admin

### 📝 邀请系统

#### 邀请配置选项

- **有效期限**：1天 / 7天 / 30天 / 永不过期
- **使用次数**：1次 / 10次 / 无限制
- **默认角色**：新成员加入后的角色（Member/Admin）

#### 邀请流程

```
创建邀请 → 生成邀请链接/二维码 → 分享给新成员 → 接受邀请 → 验证并加入
```

### 🔄 资源同步机制

1. **浏览资源** - 在团队面板查看所有共享资源
2. **一键安装** - 点击安装将资源同步到本地
3. **版本追踪** - 显示本地版本与最新版本的差异
4. **更新提示** - 当资源有更新时收到通知

### 🚀 快速开始

#### LAN 模式（局域网）
```
1️⃣ 进入 Settings → Team 配置
2️⃣ 选择 LAN Mode，启动本地服务
3️⃣ 创建团队，生成邀请链接
4️⃣ 其他成员输入邀请链接加入
5️⃣ 开始共享 Skills/Recipes/Extensions
```

#### Cloud 模式（云服务）
```
1️⃣ 部署 AGIME Team Server 到云服务器
2️⃣ 客户端配置云服务器地址
3️⃣ 创建团队并邀请成员
4️⃣ 跨地域协作和共享
```

### 📦 服务器部署

**云服务器部署**：
```bash
# 下载 Linux 版本
wget https://github.com/jsjm1986/AGIME/releases/latest/download/AGIME-linux-x64.tar.gz
tar -xzf AGIME-linux-x64.tar.gz

# 启动服务
cd AGIME-linux-x64/resources/bin
./agimed agent

# 服务运行在 http://your-server:3000
```

**Systemd 服务配置**：
```ini
[Unit]
Description=AGIME Team Server
After=network.target

[Service]
Type=simple
User=ubuntu
WorkingDirectory=/data/agime
ExecStart=/data/agime/bin/agimed agent
Restart=always

[Install]
WantedBy=multi-user.target
```

## 🏗️ 技术架构

### 本地优先
AGIME 完全运行在你的电脑上，所有数据处理都在本地完成，敏感信息永不外传。

### 通用模型兼容
支持市面上几乎所有主流 AI 模型，包括：
- **云端模型** - 通过 API 接入各大模型服务商
- **本地模型** - 通过 Ollama 等方案完全离线运行

> 💡 你可以根据任务需求和隐私要求，自由选择最适合的模型。

### MCP 扩展协议
基于 Model Context Protocol 标准，支持丰富的插件生态：
- 🔧 **工具扩展** - 文件操作、命令执行、网页浏览
- 🔗 **服务集成** - 连接各类外部服务和 API
- 🎨 **自定义能力** - 开发专属插件满足个性化需求

### 安全沙箱
所有操作在受控环境中执行，支持多级权限管理，确保系统安全。

### 跨平台支持
原生支持 Windows、macOS、Linux，统一的用户体验。
