# AGIME 核心功能

AGIME 的核心在于 **全栈自动化** 与 **无限扩展性**。它不仅仅是一个对话框，而是一个连接了操作系统、文件系统和互联网的智能中枢。

---

## 1. 自动化 (Automation)

AGIME 能够直接控制计算机执行任务，而非仅仅给出建议。

### 文档批量处理
基于视觉模型 (Vision-Language Models)，AGIME 能像人眼一样阅读 PDF、图片和扫描件。
- **场景**: 提取发票关键信息、合同条款比对、简历筛选。
- **能力**: 支持 OCR、版面分析、表格提取。

### 系统级操作 (Computer Use)
安全地控制鼠标和键盘，模拟人类操作。
- **场景**: 打开 ERP 软件填表、在 Photoshop 中执行批处理、管理系统后台。
- **机制**: 基于 Accessibility API 和视觉反馈闭环，确保操作准确。

### 定时任务调度
内置 Cron 级调度器，支持自然语言配置。
- **指令**: *"每天早上 9 点把最新的行业新闻汇总发给我"*
- **指令**: *"每周五下午 5 点备份 Download 文件夹到 NAS"*

---

## 2. 记忆与知识 (Memory & Knowledge)

### 长期记忆 (Long-term Memory)
AGIME 会自动记录你的工作习惯、偏好和常用指令。
- **偏好学习**: 自动记住输出格式（如 "总是用 Markdown 表格回复"）。
- **任务上下文**: 记住上周的项目背景，无需重复输入。

### 本地知识库 (Local RAG)
无需上传云端，直接索引本地文件夹。
- **零配置**: 拖入文件夹即可建立索引。
- **隐私**: 向量数据库 (Vector DB) 部署在本地，数据不出内网。

---

## 3. 扩展系统 (MCP Ecosystem)

AGIME 完整支持 **Model Context Protocol (MCP)**，这是连接 AI 与外部世界的标准协议。

### 官方插件
| 插件 | 功能 |
|:---|:---|
| **FileSystem** | 安全读写本地文件和目录 |
| **Browser** | 控制浏览器进行网页抓取和自动化 |
| **Terminal** | 执行 Shell/PowerShell 命令 |
| **Database** | 连接 MySQL/PostgreSQL 进行数据分析 |

### 自定义开发
企业可使用 Python/Node.js 开发私有 MCP 服务器，对接内部 API。
- **案例**: 对接公司内部的 CRM 系统，让 AGIME 直接查询客户资料。
- **案例**: 对接运维监控平台，实现自然语言查询报警日志。
